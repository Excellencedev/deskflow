Detailed Description

XDG Desktop Portal clipboard interface wrapper using QDbus.

This class provides a high-level interface to the XDG Desktop Portal clipboard functionality using QDbus for D-Bus communication. It provides async/sync conversion for clipboard operations and supports Gnome and KDE desktop environments.
